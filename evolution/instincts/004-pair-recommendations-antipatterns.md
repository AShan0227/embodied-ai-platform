# Pair Recommendations with Anti-Patterns

For board-facing strategy docs and recommendation sections:
- Pair every strategic recommendation with a short anti-pattern table
- Add source-mapping appendix in the first finalization pass (not as afterthought)
- Close with a source-accessibility audit so reviewers can separate thesis risk from citation risk
